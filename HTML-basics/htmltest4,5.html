Block element

Block element starts in a new line, it takes the complete width

Examples of block element are div, section, p, h1-h6

Inline element

Inline element takes oly the space of the content, it is used in inline styling

Examples are span, strong, em, i,

CSS selectors

Selector Uses Symbol
universal selector It selects the entire page *
Class selector It selects the element with the class name can be reused .
id selector Id name must be unique,it must not repeat #
tag selector It selects the element using the tags p, div,....
group selector To select multiple elements ,
Child selector Selects the main child of the parent >
Descendant selector Selects element inside another element
General sibling selector Selects all the child ~
adjacent sibling selector Selects adjacent child +
pseudo selector Selects inorder to hover ::
attribute selector Selects the input field []
